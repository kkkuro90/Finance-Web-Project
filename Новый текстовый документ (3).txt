В чём плюсы ASP.NET Core:
В фреймворке есть авторизация, конечные точки(/users/1, где 1 - конечная точка), обработка запросов, генерация ответа, работа с пользовательскими данными.
Полезности: конфигурации, логирование
Из чего состоит, HTTP Api's, SignalR, gRPC. Middleware для роутинга, кэширования, сжатия данных и т.п.

SignalR полезен для real time взаимодействий(например, онлайн чат)
gRPC для передачи данных(микросервисы)

Сервер - Kestrel, т.к. кроссплатформенность.
